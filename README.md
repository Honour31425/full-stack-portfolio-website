## ğŸŒ Deployment on Netlify

### Method 1: Drag and Drop
1. Create a ZIP file of the project folder
2. Go to [Netlify](https://www.netlify.com/)
3. Sign up or log in
4. Drag and drop the folder to Netlify's deployment zone

### Method 2: GitHub + Netlify (Recommended)
1. Push this code to a GitHub repository
2. Connect your GitHub account to Netlify
3. Select the repository
4. Netlify will automatically detect it as a static site
5. Click "Deploy site"

### Method 3: Netlify CLI
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Navigate to project directory
cd portfolio-3d

# Deploy
netlify deploy --prod
```
